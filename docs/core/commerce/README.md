# IsoStack Commerce Core Documentation

Commerce Core is a reusable platform lane and a sibling of product modules such as FUND.

Documentation lifecycle:

```text
00-roadmap-control
01-cr-inputs
02-triage
03-slice-planning
04-implementation-confirmations
05-review-and-test
```

Executable slices must progress through planning, implementation confirmation and
review/test in that order. Architecture decisions in `02-triage` may govern several slices
but are not themselves implementation confirmations.
