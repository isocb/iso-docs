


# Sub-processors

**Isoblue Limited – IsoStack Platform**

Isoblue Limited uses trusted infrastructure providers to operate the IsoStack platform and applications including SeasonPro.

These providers process data only as necessary to deliver the service.

---

## Current Sub-processors

| Provider       | Role                                           |
| -------------- | ---------------------------------------------- |
| **Render**     | Application hosting                            |
| **Neon**       | Managed PostgreSQL database hosting            |
| **Cloudflare** | Storage, network security and content delivery |
| **Resend**     | Transactional email delivery                   |

---

## Infrastructure Overview

These providers support the platform by providing:

* secure cloud hosting
* managed databases
* secure network infrastructure
* transactional email services.

Isoblue selects providers that maintain high standards of security and operational reliability.
